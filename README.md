# Voice-to-Voice Translator

Проект представляет собой консольное приложение на Python для обеспечения двустороннего голосового перевода в реальном времени.

## Описание работы
Программа работает по принципу рации: пользователь зажимает (или нажимает) клавишу для записи своей речи, после чего система автоматически распознает текст, переводит его через нейросеть GPT-4 и озвучивает результат на языке собеседника. После каждого цикла перевода роли (языки) автоматически меняются местами.

## Зависимости и установка
Для корректной работы скрипта необходимо установить Python 3.x и набор внешних библиотек.

### Команда для быстрой установки:
```bash
pip install PyAudio SpeechRecognition keyboard gTTS playsound g4f
```

### Детальный список используемых модулей:
1. **PyAudio** — отвечает за захват аудиопотока и работу с системными звуковыми устройствами.
2. **SpeechRecognition** — обеспечивает интеграцию с сервисами распознавания речи (Google STT).
3. **g4f (GPT4Free)** — используется для бесплатного доступа к API GPT-4, что обеспечивает высокое качество перевода.
4. **gTTS (Google Text-to-Speech)** — преобразует переведенный текст в аудиофайл.
5. **playsound** — кроссплатформенная библиотека для воспроизведения полученных аудиофайлов.
6. **keyboard** — отслеживает нажатия клавиш на клавиатуре для управления процессом записи.

## Инструкция по использованию

1. **Запуск:** Запустите скрипт через терминал.
2. **Настройка:** При запуске программа предложит просмотреть список доступных языковых кодов. Введите коды языков для обоих участников (например, `ru-RU` и `en-US`).
3. **Запись:** Следуйте подсказкам на экране. Когда появится надпись "Говорит человек", начните говорить.
4. **Завершение фразы:** Нажмите `Пробел`, чтобы остановить запись и отправить текст на перевод.
5. **Прослушивание:** Скрипт озвучит перевод, после чего автоматически предложит начать запись для второго участника на другом языке.
